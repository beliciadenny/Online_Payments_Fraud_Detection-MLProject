# Online Payment Fraud Detection System

## Description
Developed an Online Payment Fraud Detection System using Decision Tree and Random Forest classifiers to identify fraudulent transactions based on transaction type, amount, and balance changes.

## Models Used
- Decision Tree Classifier
- Random Forest Classifier

## Evaluation Metrics
- Confusion Matrix
- Classification Report
- Feature Importance

## Dataset
The dataset is not included due to size constraints.
It can be downloaded from a public source such as Kaggle.
After downloading, place it locally and update the file path in the code.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Plotly
